# Redis-trib的常用命令

```
01) create      : 创建集群
02) check       : 检查集群
03) info        : 查看集群信息
04) fix         : 修复集群
05) reshard     : 在线迁移slot
06) rebalance   : 平衡集群节点slot数量
07) add-node    : 将新节点加入集群
08) del-node    : 从集群中删除节点
09) set-timeout : 设置集群节点间心跳连接的超时时间
10) call        : 在集群全部节点上执行命令
11) import      : 将外部redis数据导入集群
```
