# 该仓库拉取代码时出现过的问题

```
# 家里和工作地方都在master分支修改更新笔记
# 工作的地方更新笔记后(git push)，在家里拉代码没有任何问题；但是家里更新笔记后，工作地方git pull代码会有问题，提示如下：
# 问题出现了很多次，每次解决后，家里一更新代码，工作的地方git push更新代码，这个问题就会出现。已多次出现
```
